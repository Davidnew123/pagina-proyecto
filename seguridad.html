Normas de seguridad

Una de las normas más importantes hablando de equipos de computación es saber el manejo de cada una de sus partes (hardware) y como funciona cada una de ellas. Esta página te dara algunos consejos importantes para el uso correcto de tu equipo de cómputo, ya sea de escritorio o portátil, así como un video sobre el mantenimiento preventivo.

Las normas basicas de seguridad para lo que es hardware es tener en un espacio apto para tenerlo un lugar firme, tener una buena Conexión electrica, tener equipo de limpieza. tener herrramientas para mantenimiento, saber como limpiar cada una de las partes del equipo y poderlas dejar funcionando de la mejor manejo del mismo.
